Designing a website for coding courses

The website includes learning paths focused on coding. The user knows what they should learn to gain a certain IT knowledge.

This project was created for the UX Design Google Certification Course on Coursera.

The Challenge

People don’t know where to start and what they should learn to start their careers in IT. Users can find a lot of courses on the internet but it’s difficult to find learning paths.

The Goal

Provide structured courses and learning paths to users who want to learn coding or programming.
