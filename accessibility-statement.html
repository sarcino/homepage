Accessibility Statement

This website fully adheres to the European Accessibility Act and complies with the Web Content Accessibility Guidelines (WCAG) 2.2, ensuring an inclusive and accessible digital experience for all users.

Date of the audit

This statement was established on 20 June 2025.

Accessible design

I use easy-to-read fonts and ensure sufficient color contrast that meets the  enhanced WCAG AAA ratio of 7:1 for better readability.

The website maintains a clear semantic structure with only one H1 heading, and proper heading levels are followed consistently throughout the page to support smooth navigation.

Text can be enlarged  without assistive technology up to 200% without losing content or functionality, ensuring users don't need to horizontally scroll to read a line on a full-screen window.

Keyboard navigation and assistive technology

A skip navigation link was added to allow users to bypass the menu and jump directly to the main content using the tab key.

The default human language is set for each web page, helping screen readers apply the correct pronunciation rules and ensuring accurate display of characters in visual browsers.

Descriptive link texts are used, along with ARIA labels to provide additional and more precise context where needed.

Several illustrations were created in Adobe Illustrator and exported as accessible SVG files. Each SVG includes a title element, offering a clear and brief description for screen readers and improving overall accessibility.
