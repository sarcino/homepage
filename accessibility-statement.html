Accessibility Statement

This website conforms to the Web Content Accessibility Guidelines (WCAG) 2.2 at Level AA and adheres to the requirements of the European Accessibility Act, ensuring an inclusive digital experience for all users.

This statement was last updated on 20 April 2026.

Accessible design

Color contrast: I utilize easy-to-read fonts and ensure color contrast meets or exceeds the WCAG AAA ratio of 7:1 for body text, providing maximum readability for users with low vision.

Semantic structure: The website maintains a clear semantic structure with only one H1 heading, and proper heading levels are followed consistently throughout the page to support smooth navigation.

Flexible viewing: All content supports browser zooming up to 200% without loss of functionality or the need for horizontal scrolling.

Keyboard navigation and assistive technology

Bypass navigation: A Skip to Content link is integrated, allowing keyboard users to jump directly to the main content using the Tab key.

Language identification: Each page specifies a default language attribute in the HTML markup. This ensures screen readers apply correct pronunciation rules and browsers render special characters accurately.

Descriptive link destinations: I use descriptive link text that conveys purpose out of context. Where visual cues are insufficient, ARIA labels provide additional precision for assistive technologies, ensuring all users understand the destination of a link.
